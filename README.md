## Spring Boot Hello World

### Technologies
Spring Boot, MVC


### Steps
##### Build and Run Application
via spring-boot-maven-plugin:
*mvn spring-boot:run*

##### Build jar, Run jar
*mvn package*  
*java -jar target/jar-name.nar*

##### Access Application
*http://localhost:8080/greeting*
